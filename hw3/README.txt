Example program displays a model airplane, loosely based
on the Piper J-3 Cub, flying in circles above two other
smaller airplanes arranged above a green polygon
representing the ground.

Controls:

- right arrow: rotate view to the right
-  left arrow: rotate view to the left
-    up arrow: rotate view up
-  down arrow: rotate view down

Attributions:
 - this program borrows heavily from ex8.c, presented in class, for supporting functions such as special(), reshape(), and Print(), Sin(), Cos(), and Idle().
